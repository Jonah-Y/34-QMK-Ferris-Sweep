This is intended for a 34 key Ferris Sweep with an RP2040 microcontroller.

There are 4 togglable layers that are shown in the pictures, except this code has qwerty for the first layer.
It is updated to use the latest QMK APIs as of 2/21/2026.

Compile and flashing instructions can be found at:
https://github.com/mechboardsguides/flashing-rp2040-promicro/tree/main
